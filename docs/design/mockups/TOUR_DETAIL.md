## Component Specifications

### Image Gallery
- **Main Image**: 1200x800px, responsive
- **Thumbnails**: 150x100px, 4 visible + counter
- **Lightbox**: Full-screen on click
- **Navigation**: Arrow keys + swipe gestures

### Booking Widget (Sticky)
- **Width**: 380px desktop, full width mobile
- **Position**: Sticky top: 100px (below nav)
- **Background**: White, shadow-lg
- **Border Radius**: 1rem
- **Padding**: 2rem

### Tabs
- **Active**: Emerald border-bottom, bold text
- **Inactive**: Zinc-400 text
- **Hover**: Emerald/30 background
- **Mobile**: Horizontal scroll, swipe gestures

### Review Cards
- **Avatar**: 48x48px circle
- **Spacing**: 1.5rem padding
- **Border**: 1px zinc-200
- **Background**: White, hover: zinc-50

### Price Display
- **Large**: text-3xl, font-bold
- **Breakdown**: Table layout, aligned right
- **Total**: Border-top, emphasized

## Accessibility

- Gallery keyboard navigable
- Booking form properly labeled
- Reviews have semantic structure
- FAQ accordion keyboard accessible
- Skip links for major sections
