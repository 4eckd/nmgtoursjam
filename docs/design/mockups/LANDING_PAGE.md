## Component Specifications

### Hero Section
- **Height**: 80vh (desktop), 70vh (tablet), 60vh (mobile)
- **Background**: Video overlay with gradient (black/70 opacity)
- **Text Color**: White
- **CTA Button**: Emerald (#10b981), hover: lighter emerald
- **Typography**:
  - H1: 3.5rem (desktop), 2.5rem (tablet), 2rem (mobile)
  - Subtitle: 1.25rem

### Tour Cards
- **Aspect Ratio**: 4:3 for images
- **Border Radius**: 0.75rem (12px)
- **Shadow**: 0 4px 6px rgba(0,0,0,0.1)
- **Hover**: Lift effect (translateY -4px)
- **Spacing**: 2rem gap between cards

### Grid Layout
- **Desktop**: 3 columns (gap: 2rem)
- **Tablet**: 2 columns (gap: 1.5rem)
- **Mobile**: 1 column (gap: 1rem)
- **Container**: Max 1400px, padding 2rem

### Color Usage
- **Primary CTA**: bg-emerald-400 hover:bg-emerald-300
- **Secondary CTA**: border-white text-white hover:bg-white/10
- **Card backgrounds**: bg-white text-zinc-900
- **Section backgrounds**: Alternate white and black/80

### Accessibility
- All images have alt text
- Buttons have ARIA labels
- Min touch target: 44x44px
- Color contrast: WCAG AA compliant
- Keyboard navigation fully supported
